# Silksong Map Data
Raw map data for [SilkSong Interactive Map](https://scripterswar.com/silksong/map).

# How to contribute
1. Contact @RainingChain on [Discord](https://discord.gg/dKEegX4).
1. On https://scripterswar.com/silksong/map website, click the Wrench icon and select a contributor mode (Icon or links).
1. Place icons/map links on the website
  - Check the folder `icons` for the list of possible `iconUrl`
1. Copy-paste the icon/map links positions to modify `silksong_map_data.json`.
1. Publish a PR with your modifications.
